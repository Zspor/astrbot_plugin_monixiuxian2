# AstrBot 插件：文字修仙（简化版）

**版本:** 2.4.1-lite

一款基于 AstrBot 设计的**简化版**文字修仙游戏插件。只保留最核心的功能：创建角色和查看信息。

## 📌 简化版说明

本版本是原完整版插件的简化版，移除了以下功能：
- ❌ 签到系统
- ❌ 闭关修炼系统
- ❌ 突破境界系统
- ❌ 商店/背包/装备系统
- ❌ 宗门系统
- ❌ 战斗系统（PvP、世界Boss）
- ❌ 秘境探索系统
- ❌ 银行理财系统
- ❌ 道号系统

**仅保留功能：**
- ✅ 创建角色（随机灵根）
- ✅ 查看角色信息

## 快速开始

*(确保���的 AstrBot 实例已正确加载此插件)*

1.  **开启你的修仙之路**: 在任意聊天窗口（QQ群或私聊），发送：
    `我要修仙`

2.  **查看角色信息**:
    发送 `我的信息` 查看你的角色数据。

3.  **获取帮助**:
    发送 `修仙帮助` 查看可用命令。

## 游戏指令

| 功能 | 指令 | 说明 |
| :--- | :--- | :--- |
| **开启仙途** | `我要修仙` | 创建你的专属修仙角色，每个ID仅可创建一次。 |
| **查看信息** | `我的信息` | 查看你当前的境界、灵根、修为、灵石等详细信息。 |
| **获取帮助** | `修仙帮助` | 显示帮助信息。 |

## 配置文件说明

本插件所有配置均在 `_conf_schema.json` 文件中，无需改动代码。

* **`ACCESS_CONTROL.WHITELIST_GROUPS`**: 群聊白名单配置，留空表示所有群聊都可用。
* **`VALUES.INITIAL_GOLD`**: 新玩家创建角色时获得的初始灵石数量。
* **`SPIRIT_ROOT_SPEEDS`**: 各种灵根的修炼速度倍率配置。
* **`SPIRIT_ROOT_WEIGHTS`**: 各种灵根的抽取权重配置。
* **`FILES.DATABASE_FILE`**: 数据库文件名（默认：xiuxian_data_lite.db）。

### 白名单配置示例

如果您希望插件只在特定群聊中生效，可以在AstrBot的插件配置界面中配置白名单：

1.  进入插件配置页面
2.  找到"访问控制"配置项
3.  在"白名单群号列表"中添加允许使用的QQ群号
4.  留空则表示所有群聊都可用

## 灵根系统

本版本保留了完整的17种灵根系统：

### 17种灵根类型
- **废柴系列**: 伪灵根（0.5x速度）
- **多灵根系列**: 四灵根、三灵根、双灵根（0.6x - 0.9x速度）
- **五行单灵根**: 金、木、水、火、土（1.0x速度）
- **变异灵根**: 雷、冰、风、暗、光（1.25x - 1.3x速度）
- **天灵根**: 天金、天木、天水、天火、天土、天雷（1.5x速度）
- **传说级**: 阴阳灵根、融合灵根（1.8x速度）
- **神话级**: 混沌灵根（2.0x速度）
- **禁忌级**: 先天道体、神圣体质（2.3x - 2.5x速度）
